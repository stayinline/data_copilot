# AI Data Copilot — 图片生成提示词汇总

> 对应PPT：`AI-Competition-Presentation-V3.md`
> 全局风格：科技竞赛风、深蓝主色调、数据可视化质感、干净利落的现代UI

---

## 全局风格定义

| 维度 | 规范 |
|------|------|
| **色彩体系** | 主色：深蓝 `#0A1628`、电光蓝 `#00D4FF`；辅助色：紫色 `#7B61FF`、绿色 `#00E676`、橙色 `#FF6D00`、红色 `#FF3D71` |
| **字体风格** | 无衬线粗体标题，数字用等宽或科技字体，关键数据加粗放大 |
| **视觉语言** | 渐变背景、发光线条、半透明卡片、圆角模块、图标+文字组合 |
| **氛围** | 竞赛级路演感、信息密度高但不杂乱、有"降维打击"的视觉张力 |
| **比例** | 16:9 宽屏，适配PPT全屏展示 |
| **留白** | 关键金句周围大量留白，形成视觉聚焦 |

## 使用建议

1. **AI绘图工具：** 使用 wan2.7-image-pro 通过 DashScope API 生成
2. **文字叠加：** 架构图、矩阵图、列表类（S3-S11、S16-S17）的文字建议用 PPT 后期叠加，AI 生成的文字通常不够精准
3. **Demo截图：** S12-S15 使用真实平台截图，不需要 AI 生成
4. **全图生成：** S1（封面）、S2（数据高墙）可全图生成，文字质量要求高
5. **一致性：** 建议先生成 S1 封面和 S7 架构作为风格锚点，后续图片参照统一色调
6. **执行脚本：** `python scripts/generate_ppt_images.py`（生成全部），或指定 slide ID
